## Field Types Legend

- **text**: Free-form text input (names, numbers, addresses, etc.)
- **date**: Single date picker (birth dates, etc.)
- **expiry_date**: Date picker with expiry tracking (certifications that expire)
- **boolean**: Yes/No or Completed/Not Completed
- **number**: Numeric value

## Document Requirements

Fields marked "Requires Document: Yes" will show a file upload option in the UI, allowing users to attach certificates, licenses, or other supporting documentation.

## Expiry Tracking

Fields with type **expiry_date** will:
- Show visual warnings when expiring within 30 days
- Display as expired when the date has passed
- Be included in expiry reports and notifications (Phase 2)

## Usage in UI

When adding a competency, users will:
1. Select a category (e.g., "NDT Certifications")
2. Select a specific competency (e.g., "EN 9712 PAUT L2")
3. The form dynamically shows the appropriate input type
4. If required, a document upload field appears
5. For expiry dates, both the cert number and expiry date can be entered
