# Inbox

Use this folder for raw notes before they are sorted into plans, decision records, implementation docs, or reference pages.

## Triage

- Does this belong in an existing doc?
- Should it become a dated plan under `docs/plans`?
- Is it a durable architectural choice that needs a decision record?
- Is it just session context that can stay in a daily note?
